# Digital Time Glow:

Digital Time Glow is a simple digital clock project built using basic HTML, CSS, and JavaScript.  
The project focuses on understanding DOM manipulation, time functions, and interval handling in JavaScript.


# About The Project:

This project uses:

- A simple HTML structure with a `div` and `p` tag inside the body.
- CSS for basic styling and glow effect design.
- JavaScript to fetch the paragraph element using `getElementById`.
- A function called `currentTime()` to display real-time time.
- `setInterval()` to update the time every second.
- A time object to store the current time values.
- `setTimeout()` used for interval control.


# Features:

- Real-time digital clock
- Glow styling using CSS
- DOM manipulation using JavaScript
- Clean and beginner-friendly code structure
